ESTOF STAFF - GitHub build

1. Apri questo ZIP sul computer
2. NON caricare lo ZIP su GitHub
3. Carica su GitHub tutti i file e cartelle che ci sono dentro
4. Controlla che esista la cartella .github/workflows
5. Vai su Actions
6. Apri Build ESTOF STAFF APK
7. Clicca Run workflow
8. Scarica l'artifact estof-staff-apk
9. Dentro trovi app-debug.apk
